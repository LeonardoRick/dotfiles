## Tool Tips

- Chrome DevTools MCP replaces Crawl4AI for reference gathering — more visual, already installed
- Ideogram MCP for logos and detailed images; Gemini native image gen for fast iteration on heroes/backgrounds
- Neither AI image tool produces production-ready logos — vector refinement always needed
- Animatopy, Google Whisk, Cosmos.so are browser-only — cannot be used by the agent directly
- SuperDesign MCP is too early-stage (68 stars, 10 commits) — dropped from workflow
